Quel est l'objectif du projet ?

Notre projet est simple: un photoshop intelligent, permettant d'aider les personnes qui étudient ou travaillent à EPITA !

Derrière le doux nom d'Epipaintor résidera un utilitaire, permettant de dessiner ou modifier une image. L'idée est d'implémenter tout un éventail de fonctionnalités graphiques, utilisant chacunes leur algorithme propre, sur le modèle du logiciel Photoshop.

Notre logiciel permet à n'importe qui de pouvoir dessiner simplement, d'appliquer des filtres, de pour retoucher des photos, retirer des éléments...

C'est un formidable outils qui peut être utile pour les étudiants d'EPITA par exemple, pour faire des croquis lors de révision. Il est simple d'utilisation et très intuitif.

Avec notre ligiciel il sera possible :

  • Dessiner avec différents outils (pinceau, seau, gomme...)
  • Selectionner des éléments (outil baguette magique ou simple selection rectangulaire)
  • Appliquer des filtres (noir et blanc, sepia...)
  • Ajouter des formes
  • Utiliser un systeme de superposition de calques

Notre organisation de travail

Comment tout s'organise ?

  • Afin de pouvoir avancer sur le projet indépendamment les uns des autres, nous avons utilisé l'outil Git. Git permet de travailler chacun de son côté sur le projet, tout en nous donnant la possibilité de pouvoir rassembler notre travail. Il permet aussi d'éviter les potentiels problèmes liés à l'unification de nos parties avec un système de commit, push et pull.
  • En ce qui concerne la rédaction des documents nous utilisons Overleaf . Ce site nous permet de pouvoir rédiger en LaTeX tous documents tel que les rapports, le cahier de charges... Chacun des membres peut donc écrire sur le document et obtenir un aperçu du PDF final instantanément. Tout comme git, c'est un outil très collaboratif.

Les librairies utilisées

GTK

GTK (The GIMP Toolkit) est un ensemble de bibliothèques permettant de réaliser des interfaces graphiques. Cette bibliothèque a été développée originellement pour les besoins du logiciel de traitement d'images GIMP.

SDL 2.0

Simple DirectMedia Layer (SDL) est une bibliothèque libre. Son API est utilisée pour créer des applications multimédias en deux dimensions pouvant comprendre du son comme les jeux vidéo, les démos graphiques, les éditeurs d'images...

SDL_image

Cette librairie permet de charger des images, d'analyser un fichier pour connaître son format d'image.
