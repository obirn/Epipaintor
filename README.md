## Requiremennts:
    - Git
    - Make
    - A Unix environment
    - Gcc

## Install:

First, clone the git repo:
```shell
    git clone git@github.com:obirn/Epinator.git
```

Then, go in the folder by doing:
```shell
    cd Epinator
```

Compile the code:
```shell
    make
```

Execute the program:
```shell
    ./epinator
```
